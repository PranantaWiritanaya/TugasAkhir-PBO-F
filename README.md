# Toko Buku XYZ

 Program manajemen penjualan buku digunakan oleh pegawai untuk melakukan pendataan manajemen took buku. Pada awal tampilan seorang admin dapat memilih login, Untuk menuju dashboard pada program manajemen penjualan took buku ini. 
	Jika seorang pegawai belum memiliki akun maka ia dapat melakukan proses registrasi terlebih dahulu pada tampilan program tersebut. Setelah selesai melakukan registrasi maka ia dapat menekan menu login untuk menuju dashboard. Pada menu dashboard terdapat beberapa pilihan menu seperti tampilan input buku, dimana pada menu ini terdapat data-data dari buku yang ada, seorang pegawai dapat menambah hingga menghapus data buku yang ada. Di dashboard juga terdapat menu pegawai yang berisi data-data karyawan, seorang pegawai dapat menambah hingga mengedit data pegawai tersebut. Selain itu pada program ini juga dilengkapi menu transaksi penjualan pelanggan untuk membantu dalam manajemen penjualan buku.
